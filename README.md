好想工作室切版練習（第三個樣版）
== 

## 使用技術與資源

純 CSS 切版。
- CSS Framework: Bootstrap 4
- RWD
- 手刻純 CSS menu bar
- Fontawesome

## 介紹：

第一次使用 `Bootstrap 4` 切版的練習，主要是練習對 `grid system` 及元件的熟悉度，了解每個斷點代表的意義（裝置的橫向與直向）。
其中利用 `counter` 及 `counter-increment` 搭配偽元素來製作粗體並且 `margin` 一致的 `list item` (@TIPS OF THE WEEK)。
